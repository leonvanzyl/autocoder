# UI Preview Images

This directory contains preview images for UI library and visual style selectors.

## Required Images

### UI Libraries
- `shadcn-button.png` - shadcn/ui button component preview (400x300)
- `shadcn-card.png` - shadcn/ui card component preview (400x300)
- `shadcn-dialog.png` - shadcn/ui dialog component preview (400x300)
- `ark-accordion.png` - Ark UI accordion component preview (400x300)
- `ark-combobox.png` - Ark UI combobox component preview (400x300)
- `ark-tooltip.png` - Ark UI tooltip component preview (400x300)
- `radix-primitives.png` - Radix UI primitives preview (400x300)

### Visual Styles
- `style-default.png` - Modern/Clean style preview (400x300)
- `style-neobrutalism.png` - Neobrutalism style preview (400x300)
- `style-glassmorphism.png` - Glassmorphism style preview (400x300)
- `style-retro.png` - Retro/Arcade style preview (400x300)

## Image Guidelines

- Format: PNG with transparency where appropriate
- Size: 400x300 pixels
- Show component or style clearly
- Use consistent backgrounds
- Optimize for web (compress without quality loss)

## Fallback Behavior

If an image is not found, the selector components will display a placeholder or icon instead.
